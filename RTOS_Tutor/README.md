# RTOS examples and guides

FreeRTOS examples for **Arduino**, **STM32F4**, and other **Cortex-M** boards, plus written guides that match the **STM32 Driver Mentor** web app.

## Contents

| Item | Description |
|------|-------------|
| **RTOS Study Guide** | Full written guide (the same material is available from the app’s Study Guides page) |
| **FreeRTOS Simulator Guide** | Walkthrough for running FreeRTOS on a PC |
| **Example firmware** | STM32-style projects covering tasks, queues, semaphores, mutexes, timers, and trace/recording |

## Documentation

Use [FreeRTOS.org](https://www.freertos.org/) for API references - especially [kernel configuration](https://www.freertos.org/a00110.html) and the [task](https://www.freertos.org/a00019.html), [queue](https://www.freertos.org/a00018.html), and [semaphore](https://www.freertos.org/a00024.html) sections.

## Web app

Open **`index.html`** in the mentor tool, choose **RTOS / FreeRTOS**, and use the interactive lessons together with hands-on firmware experiments.
